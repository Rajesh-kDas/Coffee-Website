☕ Coffee Website
🚀 Project Overview

This is a full-stack Coffee Website developed using Java Spring Boot for backend and HTML, CSS, JavaScript for frontend.

🛠️ Technologies Used
Java
Spring Boot
HTML, CSS, JavaScript
MySQL
✨ Features
View coffee menu
Place orders
Order confirmation page
Responsive UI design
▶️ How to Run
Clone the repository
Open the project in VS Code or IntelliJ
Open terminal inside project folder

Run the command:

mvn spring-boot

Open browser and go to:
http://localhost:8080
